Radarr Upcoming – Configuration

Configure the connection to your Radarr instance. The plugin will fetch your Wanted (monitored, not-yet-downloaded) movie list and display it as an Upcoming section on your Jellyfin home page.

The URL where your Radarr instance is reachable, e.g. http://localhost:7878 or https://radarr.yourdomain.com.
Found in Radarr under Settings → General → Security → API Key.
The maximum number of upcoming movies to display. Default is 20.
When enabled, movies without any known release date are hidden from the list.

Immediately fetches the latest Wanted list from Radarr without waiting for the next scheduled sync.
